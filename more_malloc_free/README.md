# C - More malloc, free

## Description
This project covers more advanced use of malloc, free and exit
for dynamic memory allocation in C programming.

## Files

| File | Description |
| --- | --- |
| `0-malloc_checked.c` | Function that allocates memory and exits with status 98 if malloc fails |
| `1-string_nconcat.c` | Function that concatenates two strings using at most n bytes of the second string |
| `2-calloc.c` | Function that allocates memory for an array and sets it to zero |
| `3-array_range.c` | Function that creates an array of integers from min to max |
