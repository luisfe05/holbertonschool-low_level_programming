# C - Variables, if, else, while

Holberton School project to learn about variables, conditional statements (`if`, `else`), and loops (`while`, `for`) in C. Includes exercises with `putchar`, `printf`, ASCII arithmetic, and limited-instruction challenges.

## Tasks

| # | File | Description |
|---|------|-------------|
| 0 | `0-positive_or_negative.c` | Prints whether a random number stored in `n` is positive, negative, or zero. |
| 1 | `1-last_digit.c` | Prints the last digit of a random number `n`, and checks if it is greater than 5, equal to 0, or less than 6 and not 0. |
| 2 | `2-print_alphabet.c` | Prints the lowercase alphabet, followed by a new line, using `putchar`. |
| 3 | `3-print_alphabets.c` | Prints the lowercase alphabet followed by the uppercase alphabet, on one line, using `putchar`. |
| 4 | `4-print_alphabt.c` | Prints the lowercase alphabet except the letters `q` and `e`, using `putchar`. |
| 5 | `5-print_numbers.c` | Prints all single-digit numbers of base 10 in ascending order, using `printf`. |
| 6 | `6-print_numberz.c` | Prints all single-digit numbers of base 10 in ascending order, using `putchar` only (no `char` variable, no `printf`). |
| 7 | `7-print_tebahpla.c` | Prints the lowercase alphabet in reverse, using `putchar`. |
| 8 | `8-print_base16.c` | Prints all the numbers of base 16 in lowercase, using `putchar`. |
| 9 | `9-print_comb.c` | Prints all single-digit numbers of base 10, separated by `, `, using `putchar`. |

## How to compile

```
gcc -Wall -Werror -Wextra -pedantic 0-positive_or_negative.c -o positive
./positive
```
