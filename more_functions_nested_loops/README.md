# C - More functions, more nested loops

Holberton School project to practice writing more functions and using nested loops in C. Includes shape drawing with `_putchar`, digit manipulation, and a classic FizzBuzz exercise.

## Tasks

| # | File | Description |
|---|------|-------------|
| — | `_putchar.c` | Helper function that writes a character to stdout using `write`. |
| — | `main.h` | Header file with the prototypes of all the functions written in this project. |
| 0 | `0-isupper.c` | Function `_isupper` that checks for an uppercase character. Returns 1 if `c` is uppercase, 0 otherwise. |
| 1 | `1-isdigit.c` | Function `_isdigit` that checks for a digit from 0 to 9. Returns 1 if `c` is a digit, 0 otherwise. |
| 2 | `2-mul.c` | Function `mul` that multiplies two integers. |
| 3 | `3-print_numbers.c` | Function `print_numbers` that prints the numbers from 0 to 9, followed by a new line. |
| 4 | `4-print_most_numbers.c` | Function `print_most_numbers` that prints the numbers from 0 to 9, except `2` and `4`. |
| 5 | `5-more_numbers.c` | Function `more_numbers` that prints the numbers from 0 to 14, ten times. |
| 6 | `6-print_line.c` | Function `print_line` that draws a straight horizontal line using `n` underscore characters. |
| 7 | `7-print_diagonal.c` | Function `print_diagonal` that draws a diagonal line using `n` backslashes. |
| 8 | `8-print_square.c` | Function `print_square` that prints a square of `size` using the `#` character. |
| 9 | `9-fizz_buzz.c` | Program that prints the numbers from 1 to 100 following the FizzBuzz rules. |
| 10 | `10-print_triangle.c` | Function `print_triangle` that prints a right-aligned triangle of `size` using the `#` character. |

## How to compile

```
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 *.c -o output
```
