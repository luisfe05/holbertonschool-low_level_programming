# C - Pointers, arrays and strings

Holberton School project that combines three tracks on pointers, arrays, and strings in C. Introduces memory manipulation, string handling, array operations, and custom implementations of standard library functions like `strcpy`, `strcat`, `strstr`, `memset`, and `atoi`.

## Pointers, arrays and strings

| File | Description |
|------|-------------|
| `0-reset_to_98.c` | Function `reset_to_98` that updates the value pointed to by a pointer to 98. |
| `1-swap.c` | Function `swap_int` that swaps the values of two integers. |
| `2-strlen.c` | Function `_strlen` that returns the length of a string. |
| `3-puts.c` | Function `_puts` that prints a string followed by a new line. |
| `4-print_rev.c` | Function `print_rev` that prints a string in reverse, followed by a new line. |
| `5-rev_string.c` | Function `rev_string` that reverses a string in place. |
| `6-puts2.c` | Function `puts2` that prints every other character of a string, starting at index 0. |
| `7-puts_half.c` | Function `puts_half` that prints the second half of a string, followed by a new line. |
| `8-print_array.c` | Function `print_array` that prints `n` elements of an integer array. |
| `9-strcpy.c` | Function `_strcpy` that copies a string from source to destination. |
| `100-atoi.c` | Function `_atoi` that converts a string to an integer. |

## More pointers, arrays and strings

| File | Description |
|------|-------------|
| `0-strcat.c` | Function `_strcat` that concatenates two strings. |
| `1-strncat.c` | Function `_strncat` that concatenates two strings using at most `n` bytes from the source. |
| `2-strncpy.c` | Function `_strncpy` that copies a string using at most `n` bytes. |
| `3-strcmp.c` | Function `_strcmp` that compares two strings. |
| `4-rev_array.c` | Function `reverse_array` that reverses the content of an array of integers. |
| `5-string_toupper.c` | Function `string_toupper` that changes all lowercase letters of a string to uppercase. |
| `6-cap_string.c` | Function `cap_string` that capitalizes all words of a string. |
| `7-leet.c` | Function `leet` that encodes a string into 1337. |

## Even more pointers, arrays and strings

| File | Description |
|------|-------------|
| `0-memset.c` | Function `_memset` that fills the first `n` bytes of a memory area with a constant byte. |
| `1-memcpy.c` | Function `_memcpy` that copies `n` bytes from one memory area to another. |
| `2-strchr.c` | Function `_strchr` that locates a character in a string. |
| `3-strspn.c` | Function `_strspn` that gets the length of a prefix substring. |
| `4-strpbrk.c` | Function `_strpbrk` that searches a string for any of a set of bytes. |
| `5-strstr.c` | Function `_strstr` that locates a substring inside another string. |
| `7-print_chessboard.c` | Function `print_chessboard` that prints an 8x8 chessboard. |
| `8-print_diagsums.c` | Function `print_diagsums` that prints the sum of the two diagonals of a square matrix. |

## How to compile

```
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 *.c -o output
```
