# C - malloc, free

## Description
This project covers the use of malloc and free in C programming
for dynamic memory allocation.

## Files

| File | Description |
| --- | --- |
| `0-create_array.c` | Function that creates an array of chars initialized with a specific char |
| `1-strdup.c` | Function that returns a pointer to a newly allocated copy of a string |
| `2-str_concat.c` | Function that concatenates two strings into a new allocated string |
| `3-alloc_grid.c` | Function that returns a pointer to a 2D array of integers initialized to 0 |
| `4-free_grid.c` | Function that frees a 2D grid previously created by alloc_grid |
