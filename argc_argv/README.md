# C - argc, argv

## Description
This project covers the use of argc and argv in C programming,
which allow programs to receive and handle command line arguments.

## Files

| File | Description |
| --- | --- |
| `0-whatsmyname.c` | Program that prints its own name followed by a new line |
| `1-args.c` | Program that prints the number of arguments passed to it |
| `2-args.c` | Program that prints all arguments it receives |
| `3-mul.c` | Program that multiplies two numbers passed as arguments |
| `4-add.c` | Program that adds positive numbers passed as arguments |
