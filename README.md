# holbertonschool-low_level_programming

Collection of Holberton School projects covering low-level programming in C.

## Projects

| Folder | Description |
|--------|-------------|
| `hello_world` | Introduction to C: compilation process, `puts`, `printf`, and data type sizes. |
| `variables_if_else_while` | Variables, conditional statements, and loops in C. |
| `functions_nested_loops` | Functions, nested loops, and header files. |
| `more_functions_nested_loops` | More functions and nested loops, including shape printing and FizzBuzz. |
| `pointers_arrays_strings` | Pointers, arrays, and strings — basic to advanced: custom implementations of `strlen`, `strcpy`, `strcat`, `strcmp`, `memset`, `strstr`, `atoi`, and more. |
| `recursion` | Introduction to recursion in C. |
| `argc_argv` | Command-line arguments in C programs. |
| `malloc_free` | Dynamic memory allocation with `malloc` and `free`. |
| `more_malloc_free` | More dynamic memory allocation and reallocation. |
| `structures_typedef` | Structures and `typedef` in C. |
| `function_pointers` | Function pointers and their uses in C. |
| `variadic_functions` | Variadic functions using `stdarg.h`. |
| `singly_linked_lists` | Singly linked lists: creation, traversal, and deletion. |
| `doubly_linked_lists` | Doubly linked lists: creation, traversal, and deletion. |
| `file_io` | File input/output using system calls. |
| `hash_tables` | Hash tables: implementation and collision handling. |
