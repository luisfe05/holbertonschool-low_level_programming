# C - Functions, nested loops

Holberton School project to learn how to write, declare, and use functions in C, along with nested loops. Introduces header files, function prototypes, and the custom helper `_putchar`.

## Tasks

| # | File | Description |
|---|------|-------------|
| — | `_putchar.c` | Helper function that writes a character to stdout using `write`. |
| — | `main.h` | Header file with the prototypes of all the functions written in this project. |
| 0 | `0-putchar.c` | Program that prints `_putchar`, followed by a new line, using the `_putchar` function. |
| 1 | `1-alphabet.c` | Function `print_alphabet` that prints the lowercase alphabet followed by a new line. |
| 2 | `2-print_alphabet_x10.c` | Function `print_alphabet_x10` that prints the lowercase alphabet ten times, each followed by a new line. |
| 3 | `3-islower.c` | Function `_islower` that checks for a lowercase character. Returns 1 if `c` is lowercase, 0 otherwise. |
| 4 | `4-isalpha.c` | Function `_isalpha` that checks for an alphabetic character. Returns 1 if `c` is a letter, 0 otherwise. |
| 5 | `5-sign.c` | Function `print_sign` that prints the sign of a number and returns 1, 0, or -1. |
| 6 | `6-abs.c` | Function `_abs` that computes the absolute value of an integer. |
| 7 | `7-print_last_digit.c` | Function `print_last_digit` that prints the last digit of a number. |
| 8 | `8-24_hours.c` | Function `jack_bauer` that prints every minute of the day from `00:00` to `23:59`. |
| 9 | `9-times_table.c` | Function `times_table` that prints the 9 times table, starting with 0. |
| 10 | `10-add.c` | Function `add` that adds two integers and returns the result. |
| 11 | `11-print_to_98.c` | Function `print_to_98` that prints all natural numbers from `n` to 98, separated by `, `, followed by a new line. |

## How to compile

```
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 *.c -o output
```
