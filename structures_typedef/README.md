# C - Structures, typedef

## Description
This project covers the use of structures and typedef in C programming
to group related data together under one type.

## Files

| File | Description |
| --- | --- |
| `dog.h` | Header file defining the struct dog type and dog_t typedef |
| `1-init_dog.c` | Function that initializes a variable of type struct dog |
| `2-print_dog.c` | Function that prints a struct dog |
| `4-new_dog.c` | Function that creates a new dog using malloc |
| `5-free_dog.c` | Function that frees a dog |
