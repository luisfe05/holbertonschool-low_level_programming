# C - Singly Linked Lists

## Description
This project covers the use of singly linked lists in C programming.
A linked list is a chain of nodes where each node holds a value
and a pointer to the next node.

## Files

| File | Description |
| --- | --- |
| `0-print_list.c` | Function that prints all elements of a list_t list |
| `1-list_len.c` | Function that returns the number of elements in a list |
| `2-add_node.c` | Function that adds a new node at the beginning of a list |
| `3-add_node_end.c` | Function that adds a new node at the end of a list |
| `4-free_list.c` | Function that frees a list_t list |
