# C - Hash Tables

## Description
This project covers the use of hash tables in C programming.
A hash table stores key-value pairs using a hash function to
compute an index into an array of buckets. Collisions are
handled using chaining with linked lists.

## Files

| File | Description |
| --- | --- |
| `0-hash_table_create.c` | Function that creates a hash table |
| `1-djb2.c` | Hash function implementing the djb2 algorithm |
| `2-key_index.c` | Function that gives the index of a key in a hash table |
| `3-hash_table_set.c` | Function that adds or updates an element in the hash table |
| `4-hash_table_get.c` | Function that retrieves a value associated with a key |
| `5-hash_table_print.c` | Function that prints a hash table |
| `6-hash_table_delete.c` | Function that deletes a hash table |
